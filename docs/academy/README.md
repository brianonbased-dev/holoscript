# HoloScript Academy

Welcome to HoloScript Academy! This comprehensive learning platform will take you from beginner to expert in creating VR/XR experiences with HoloScript.

## Course Structure

### Level 1: Fundamentals (10 lessons)

Learn the basics of HoloScript and create your first VR scenes.

| #    | Lesson                                                                                | Description                                       | Duration |
| ---- | ------------------------------------------------------------------------------------- | ------------------------------------------------- | -------- |
| 1.1  | [What is HoloScript?](./level-1-fundamentals/01-what-is-holoscript.md)                | Introduction to the language and ecosystem        | 15 min   |
| 1.2  | [Installation & Setup](./level-1-fundamentals/02-installation.md)                     | Setting up your development environment           | 20 min   |
| 1.3  | [Your First Scene](./level-1-fundamentals/03-first-scene.md)                          | Create a simple VR scene with interactive objects | 30 min   |
| 1.4  | [Understanding Compositions](./level-1-fundamentals/04-understanding-compositions.md) | Deep dive into 3D object definitions              | 25 min   |
| 1.5  | [Properties Deep Dive](./level-1-fundamentals/05-properties.md)                       | Position, rotation, scale, color, and more        | 30 min   |
| 1.6  | [Introduction to Traits](./level-1-fundamentals/06-traits-intro.md)                   | Adding behaviors with @grabbable, @physics, etc.  | 35 min   |
| 1.7  | [Basic Interactivity](./level-1-fundamentals/07-interactivity.md)                     | Event handlers: onGrab, onClick, onHover          | 30 min   |
| 1.8  | [Templates & Reuse](./level-1-fundamentals/08-templates.md)                           | Creating reusable object definitions              | 25 min   |
| 1.9  | [Project Structure](./level-1-fundamentals/09-project-structure.md)                   | Organizing larger HoloScript projects             | 20 min   |
| 1.10 | [Building & Deploying](./level-1-fundamentals/10-building.md)                         | Compiling and deploying your VR experience        | 25 min   |

## Prerequisites

- Basic programming knowledge (variables, functions, loops)
- Familiarity with any C-style language (JavaScript, TypeScript, etc.)
- A computer with Node.js 18+ installed
- Optional: A VR headset for testing

## How to Use This Course

1. **Start with Level 1** - Even experienced developers should skim the fundamentals
2. **Complete the exercises** - Each lesson includes hands-on exercises
3. **Build the projects** - End-of-level projects reinforce what you've learned
4. **Join the community** - Ask questions in our Discord server
5. **Contribute back** - Found an error? Submit a PR!

## Quick Start

```bash
# Install HoloScript CLI
npm install -g @holoscript/cli

# Create your first project
holoscript init my-first-vr-app

# Start the development server
cd my-first-vr-app
holoscript dev
```
