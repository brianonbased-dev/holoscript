# HoloScript Academy

Welcome to HoloScript Academy! This comprehensive learning platform will take you from beginner to expert in creating VR/XR experiences with HoloScript.

## Prerequisites

- Basic programming knowledge (variables, functions, loops)
- Familiarity with any C-style language (JavaScript, TypeScript, etc.)
- A computer with Node.js 18+ installed
- Optional: A VR headset for testing

## How to Use This Course

1. **Start with Level 1** - Even experienced developers should skim the fundamentals
2. **Complete the exercises** - Each lesson includes hands-on exercises
3. **Build the projects** - End-of-level projects reinforce what you've learned
4. **Join the community** - Ask questions in our Discord server
5. **Contribute back** - Found an error? Submit a PR!

## Quick Start

```bash
# Install HoloScript CLI
npm install -g @holoscript/cli

# Create your first project
holoscript init my-first-vr-app

# Start the development server
cd my-first-vr-app
holoscript dev
```
